\# Kaggle Titanic 项目



这是我的第一个 Kaggle 机器学习项目。



\## 项目目标

预测泰坦尼克号乘客的存活情况。



\## 技术栈

\- Python 3.x

\- Pandas

\- Scikit-learn

\- Jupyter Notebook



\## 模型与结果

\- 决策树 (Decision Tree) + 随机森林 (Random Forest)

\- 本地验证集准确率: \*\*79.33%\*\*

\- Kaggle 公开分数: \*\*0.78229\*\*



\## 文件说明

\- `Titanic.ipynb`: 完整的分析与建模流程

\- `train.csv` / `test.csv`: 原始数据

\- `my\_first\_submission.csv`: 最终提交文件
